How does it work?

Once your data has been entered, it is automatically saved into a convenient templating format. This format is known as JSON (JavaScript Object Notation):

      
        "basics": {{ '{' }}
          "name": "Richard Hendriks",
          "label": "Programmer",
          "email": "richard.hendriks@gmail.com",
          "phone": "(912) 555-4321",
          "website": "http://richardhendricks.com"
          {{ '}' }}
